My First Article

There is text in this article!

Since this is an HTML article, it has a good amount of extensibility. Notice there's no head or body tags for this, just an article tag! Any metadata, like tags and search-functionality, can be handled by the API. The styling can be handled by any underlying frontend.

In this way, we have a complete and functional CMS based on MongoDB and GitHub! Since blog posts are created and stored using git, any post modifications get archived automatically.

I could have used markdown for this purpose as well, but I've opted to use HTML as it is more "core" to the ecosystem. I can easily customize the structure of my blog posts in this way without any markdown transpilers and so forth.
